# @chrisaguilar/tslint-config

## Usage

`$ yarn add -D typescript tslint @chrisaguilar/tslint-config`

or

`$ npm install -D typescript tslint @chrisaguilar/tslint-config`

Then, in `tslint.json`:
```json
{
  "extends": [
    "@chrisaguilar/tslint-config"
  ],
  "rules": { "..." }
}
```
